发表于2023-08-05|更新于2025-06-30|技术教程
|总字数:346|阅读时长:1分钟|浏览量:

前面有分享一个 国内加速访问GitHub的几种方法（一） 的教程，这里分享几个开源使用的 GitHub 加速访问的软件。

简介

GitHub 访问慢的根本原因就是 CDN 内容受到 DNS 污染，无法连接使用 GitHub 的加速分发的任务，这也导致在国内访问时会出现时好时坏的情况。

在这里分享几个开源的 GitHub 镜像加速器，这里没有进行测试，又可能部分地区没法使用另一个软件这也是有可能实现加速的，所以这里我去 GitHub 上找了三个开源的 GitHub 加速神器。

GitHub 下载链接

FastGithub：https://github.com/dotnetcore/FastGithub

fetch-github-hosts：https://github.com/Licoy/fetch-github-hosts

Ghips：https://github.com/aardio/Ghips

这三款 GitHub 网速优化工具支持 Linux/Windows/mac 系统，实现 GitHub 加速，这样也在你没办法科学上网时，访问 GitHub 速度慢的窘境。
