Windows系统内存清理工具

发表于2023-09-28|更新于2025-07-28|实用工具
|总字数:544|阅读时长:1分钟|浏览量:

简介

这是一款200多kb大小且免费的 RAM 清理工具，使用本机 Windows 功能来清理内存区域。有时，程序不会释放分配的内存，从而使计算机变慢。也就是说，当您使用 Windows Memory Cleaner 来优化内存时，您可以继续工作，而不必浪费时间重新启动系统。

开源下载地址：https://github.com/IgorMundstein/WinMemoryCleaner

🚀特点

自动优化

  • Every X hours- 优化将按时间段运行
  • When free memory is below X percent- 如果可用内存低于指定百分比，优化将运行

紧凑模式

  • 箭头旁边的最小化按钮（向上/向下）可折叠和展开窗口

内存区域

  • Clean Combined Page List- 只有当启用页面组合时，才有效地从组合页面列表中刷新块
  • Clean Modified Page List- 从修改的页面列表中刷新内存，将未保存的数据写入磁盘页面移动到备用列表
  • Clean Processes Working Set- 从所有用户模式和系统工作集中删除内存，将其移动到备用或修改的页面列表。请注意，当进程运行时，- 任何代码都必须填充其工作集来执行此操作
  • Clean Standby List- 将页面从所有备用列表刷新到空闲列表
  • Clean Standby List (Low Priority)- 将页面从优先级最低的备用列表刷新到空闲列表
  • Clean System Working Set- 从系统存储集中工作删除内存

在优化外部流程中进行修复

  • 您可以构建一个进程列表，以便在内存优化时忽略

优化热键（全局）

  • CTRL + ALT + M (Customizable)- 优化

设置

  • Always on top- 将窗口固定到所有窗口的顶部
  • Auto update- 保持应用程序最新
  • Close after optimization- 优化后关闭应用程序
  • Close to the notification area- 单击关闭 (X) 按钮时将应用程序最小化到系统托盘
  • Run on startup- 系统启动后运行应用程序
  • Show optimization notifications- 优化后向通知区域发送消息
  • Start minimized- 该应用程序将最小化到系统托盘启动。单击/图标即可恢复
