实现bilibili在PotPlayer上播放

发表于2023-09-16|更新于2025-06-19|技术教程
|总字数:602|阅读时长:1分钟|浏览量:

无意中找到一个可以在PotPlayer上播放bilibili的方法。

过程

  1. 去GitHub仓库把项目git clone下来

  2. 将项目 Media/UrlList 路径下的 MediaUrlList - Bilibili.as 和 MediaUrlList - Bilibili.ico 两个文件复制到 {PotPlayer 安装路径}\Extension\Media\UrlList 文件夹下。

  3. 将项目 Media/PlayParse 路径下的 MediaPlayParse - Bilibili.as 和 MediaPlayParse - Bilibili.ico 两个文件复制到 {PotPlayer 安装路径}\Extension\Media\PlayParse 文件夹下。

  4. MediaUrlList - Bilibili.as 提供了列出 Bilibili 常用链接的功能，使用方式为按下 ctrl + U 并选择要播放的项目（也可以左上角点击打开，找到打开链接），如下图所示：

  5. 获取Cookies，点击键盘F12或者鼠标右键->检查，进入开发者工具，点击网络/NetWork选项卡，在数据包中找到 heartBeat 或 webHeartBeat，点击找到请求头中的 cookie 项，并复制保留（图中浅蓝色部分）。

  6. 打开 PotPlayer，按 F5 打开选项（也可以点击左上角找到选项），点击扩展功能下的媒体播放列表/项目，再点击 Bilibili，然后打开账户设置，在 Cookie 一栏粘贴你的 Bilibili Cookie。

  7. 点击测试按钮，如果弹出你的账号信息，就说明登录成功。

使用方法

按 ctrl + U，在文件地址列表中选择搜索，然后到上面的输入框中替换关键词，最后回车即可

跳过片头片尾

对于一些电视剧、番剧，能够跳过片头和片尾。具体设置为：在 PotPlayer 上点击鼠标右键，选择播放-跳略播放-跳略播放设置

勾选跳略播放和章节名称，并在名称列表中追加哔哩哔哩-片头和哔哩哔哩-片尾两项，每一项之间用英文分号;隔开。

列表显示缩略图

按 F6 打开播放列表，点击鼠标右键，点击样式，选择显示缩略图，即可显示视频的缩略图。

创建自动更新的播放列表

按 F6 打开播放列表，点击新建专辑，起一个合适的专辑名称，选择外部播放列表，并填写相应的链接，再点击确定即可。这样就得到一个可以自动更新的列表。

扩展

安装一个油猴插件，直接可以跳转到PotPlayer，不用手动输入地址。
