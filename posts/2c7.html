随机图API接口分享及预览

发表于2023-06-01|更新于2025-09-20|干货资源
|总字数:1.5k|阅读时长:5分钟|浏览量:

LoliAPI：https://www.loliapi.com/

  • 有自适应二次元图片，手机端和PC端，随机二次元头像等。

双端二次元自适应：https://www.loliapi.com/acg/

二次元头像：https://www.loliapi.com/acg/pp/

二次元随机图片API：https://t.mwm.moe/

  • 使用很简单，直接调用即可，而且这里的图大部分都是精品。

双端二次元自适应：https://t.mwm.moe/ycy

樱道随机图片API接口：https://img.r10086.com/

  • 樱道这里有非常多的系列可以选择，包含了二次元与三次元

  • 二次元动漫综合系列、赛马娘系列、东京食尸鬼系列、Fate系列、为美好世界献上祝福系列、某科学的超电磁炮系列、原神系列、我的世界系列、神奇宝贝系列、龙珠系列、罪恶王冠系列、鬼灭之刃系列、火影忍者系列、海贼王系列、进击的巨人系列、从零开始的异世界生活系列、刀剑神域系列、钢之炼金术师系列、妖精的尾巴系列、缘之空系列、东方project系列、猫娘系列、风景系列、物语系列、少女前线系列、明日方舟系列、重装战姬系列、P站系列、CG系列、守望先锋系列、王者荣耀系列、少女写真系列、橘里橘气系列等

  • 分的非常详细，这里还有ppt类型的加载方式，非常的不错。

随机图1：https://api.r10086.com/樱道随机图片api接口.php?图片系列=动漫综合1

随机图2：https://api.r10086.com/樱道随机图片api接口.php?图片系列=动漫综合2

零七生活API：https://api.oick.cn/

  • 可以加参数?type=pc或者?type=pe，表示电脑端图或者手机端图（横图和竖图）

手机端：https://api.oick.cn/random/api.php?type=pe

电脑端：https://api.oick.cn/random/api.php?type=pc

桑帛云API：https://api.caonm.net/

  • 这个里面也有一大堆接口，详情可以自己去研究。

二次元随机图：https://api.caonm.net/api/dm/index.php

魅影随机图片API：https://tuapi.eees.cc/

  • 有二次元图，三次元图，风景图和必应图，可以通过category选择，type可以选择直接302跳转图片地址或者返回xml或者返回json等。

二次元自适应随机图：https://tuapi.eees.cc/api.php?category=dongman&type=302

二次元随机图片API：https://t.lizi.moe

  • 此外还有萌图和随机二次元头像的接口，点击上方链接即可查看
  • 另外在接口后面加上/img.txt即可查看图片链接列表，非常方便。

PC端（横屏）：https://t.lizi.moe/pc

移动端（竖屏）：https://t.lizi.moe/mp

二次元风景：https://t.lizi.moe/fj

98情缘API：https://www.98qy.com/sjbz/

随机二次元壁纸：http://www.98qy.com/sjbz/api.php （翻墙）

超级小兔的随机图：https://blog.xl0408.top/index.php/blog

随机图：https://imgapi.xl0408.top/index.php

岁月小筑随机背景API：http://img.xjh.me/

  • 直接使用图片有点杂乱，各种格式的图片尺寸大小都不一样，太过于广泛，可以留着后面看有其它用处没。

岁月小筑随机图：https://img.xjh.me/random_img.php

樱花随机二次元图片API：https://www.dmoe.cc/

  • 这里不多说明，用樱花的比较多，不过近期好像有点难用了

樱花随机二次元图：https://www.dmoe.cc/random.php

缙哥哥的博客：https://www.dujin.org/19203.html

  • 收录了数千张原神图片，也有其它不同类型的图，有兴趣的可以去看看，还有特别大的那种（一张二三十兆，这里可以收藏几张来用，不建议直接用于背景）

原神图：https://api.dujin.org/pic/yuanshen/

unsplash随机图片：https://www.imyangyong.com/

PAULZZH 东方Project随机图片API：https://img.paulzzh.com/

  • 全都是东方幻想乡的图，图库数量挺多的

东方随机图：https://img.paulzzh.tech/touhou/random

保罗 API：https://api.paugram.com/

  • 白底二次元图片，有点单调，但可以适合当作纯色桌面背景之类的

随机图：https://api.paugram.com/wallpaper/

小C的每日漫图介绍页的随机图：https://pixiv.xiaoc.cn/

  • 返回为json格式，包含图片标题、pid、作者和图片链接，vertical=1表示只有竖图

随机图：https://pixiv.xiaoc.cn/v1/img/random?vertical=1

随机二次元图片API - 三秋API接口站：https://api.ghser.com/random/

  • 这个好像很多都失效了，就几个链接可以用。

横屏二次元风景随机图：https://api.ghser.com/random/bg.php

竖屏随机图：https://api.ghser.com/random/pe.php

Jitsuの随机涩图API：https://blog.jitsu.top/archives/3/

  • 这个可以指定参数，调整图片大小，选择图片类型等，详见看使用文档

随机图：https://moe.jitsu.top/api/

横屏随机图：https://moe.jitsu.top/img/?sort=pc&size=mw2048

一只小嘟嘟的萝莉涩图接口

  • 这个站没有https，只能使用用http，用https会直接404

横屏随机图：http://www.yzxdd.icu/api/

EEE.DOG随机图API：https://www.eee.dog/tech/rand-pic-api.html

  • 这个也可以加参数，指定筛选的尺寸，是否显示r18，是否显示为网页等，同时也有必应和头像的随机图接口

呓喵酱的随机图：https://api.yimian.xyz/img

Lolicon API：https://api.lolicon.app/

随机涩图

  • 这个不是直接返回图片，而是一个json信息，图片的链接在json里面。
  • 同时这个也可以加tag、r18等参数来缩小图片查找范围，很好用=w=

随机涩图：https://api.lolicon.app/setu/v2

NyanCatdaの小站

随机图：https://sex.nyan.xyz/

  • 这个也是可以返回json，也可以直接返回图片，可以筛选r18，指定作者pid，指定返回数量，指定tag关键字等等

直接调用：https://sex.nyan.xyz/api/v2/img/

Jitsuの随机涩图API：https://docs.anosu.top/ 推荐使用：https://img.jitsu.top/

  • 具体使用就看文档吧，可以返回json，可以直接返回图片，可以加tag标签、r18等参数等等。

请求参数 | API Docs

通用：https://moe.jitsu.top/img/?sort=pc&size=mw1024

302跳转：https://image.anosu.top/pixiv/direct
